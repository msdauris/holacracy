# holacracy

Holacracy management system POC built with React and Vite.

## Run locally

1. `npm install`
2. `npm run dev`

Then open `http://localhost:3000`.

## What this POC includes

- Local clickable prototype to validate UX before building in production tools
- Circle navigation, role cards, project status logic, governance records, and notifications
- Dedicated `Next Actions` tab for tactical work (tension -> ideal outcome)
- `Org Map` tab with a near real-time visual simulation (auto-refresh every 5 seconds)
- Tool mapping badges showing where production data will live

## Documentation for real-world rollout

- `IMPLEMENTATION_PLAN.md` - full setup plan for Jira Cloud + M365 + Confluence + Slack + Power BI
- `RUNBOOK.md` - operating guide for day-to-day management once live
